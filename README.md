# .dotfiles

My personal dotfiles. So it always feels like `$HOME`...

## Installation

Almost everything is automated thanks to a bunch of scripts and [dotbot](https://github.com/anishathalye/dotbot).
Just clone this repository and run the install script:

```
git clone https://github.com/martindzejky/dotfiles ~/.dotfiles
cd ~/.dotfiles
./install
```

## Manual steps

I'm using [starship](https://starship.rs) for my terminal prompts. For this to work correctly, install and set up a [Nerd font](https://www.nerdfonts.com).

My current `Brewfile` already installs my favourite font as patched `font-jetbrains-mono-nerd-font`, so just set it up in the editors and terminal apps.
